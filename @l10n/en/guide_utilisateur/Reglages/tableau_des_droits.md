## ğŸ‘ï¸ Visibility levels

Each permission also gives a different **visibility scope** over platform files.

| **Visibility** | **Explanation** |
|----------------|------------------|
| I am owner | The user can view and edit only files they own. |
| Owner is in my group | The user can view and edit all files whose owner belongs to their group. |
| All | The user can view all files on the platform. |
| None | The user has no visibility on files. |

## ğŸ§­ Owner definition

The **owner** is:
- either the person who **created** the file,

You can **change a file owner** from the **Other Actions** menu.
